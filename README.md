## Description
This web application allows users to discover their "Akan Name" based on their birthday and gender.
In Ghanaian culture, children are traditionally named after the day of the week they were born.
The application takes a user's date of birth, calculates the specific day of the week,
and provides their corresponding name in Ashanti Twi.
## Behaviour Driven Development(BDD)
This project follows a logical flow to ensure a smooth user experience:
It outlines the process of capturing the birthday and gender inputs, calculating the corresponding day of the week, and matching that day to the correct Akan name. Ultimately, it ensures the user receives a clear and accurate result based on the logic of Ghanaian naming traditions.
